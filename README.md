# Student Attendance Viewer

This is a React-based web application that displays student attendance data.

The application fetches student data from an API and assigns random attendance percentages and status (Present/Absent). Users can filter students based on their attendance status and highlight students with low attendance.

## Features
- Fetch student data from API
- Random attendance generation (50% - 100%)
- Filter by Present / Absent
- Show students with attendance less than 75%
- Highlight selected student
- Sort students by attendance

## Technologies Used
- React
- Vite
- JavaScript
- CSS
- Fetch API
